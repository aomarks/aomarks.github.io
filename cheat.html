Vim

Files

  • :w Write file
  • :wqa Write all files
  • :q Quit unless pending changes
  • :q! Quit and ignore pending changes

Basic navigation

  • h Left
  • j Down
  • k Up
  • l Right

Words

  • w Start of next word
  • W Start of next word (with punctuation)
  • b Start of last word
  • B Start of last word (with punctuation)
  • e End of next word
  • E End of next word (with punctuation)

Scrolling

  • Ctrl + e Scroll one line down
  • Ctrl + y Scroll one line up
  • Ctrl + d Scroll half screen down
  • Ctrl + u Scroll half screen up
  • Ctrl + f Scroll full screen down
  • Ctrl + b Scroll full screen up
  • zzScroll so that current line is centered

Navigate lines

  • 0 Line start
  • $ Line end
  • ^ or _ First non-blank character of line
  • g_ Last non-blank character of line

Navigate screen

  • H Screen top
  • M Screen middle
  • L Screen bottom

Navigate structurally

  • % Jump to matching (), {}, or [] pair
  • } Next paragraph/block
  • { Previous paragraph/block

Navigate file

  • gg Start of file
  • g End of file
  • :n Line n of file

Copy/Paste

  • y Copy selection
  • yy or Y Copy current line
  • y$ Copy to end of line without newline
  • d Cut selection
  • dd Cut line
  • D Cut to end of line
  • p Paste clipboard after cursor
  • P Paste clipboard before cursor

Editing lines

  • j Join next line to current, with space
  • gj Join next line to current, without space
  • cc Replace line
  • c Replace to end of line

Editing characters

  • ~ Toggle case
  • x Cut character
  • xp Transpose

Undo/Redo

  • u Undo
  • Ctrl + r Redo

Search/Replace

  • /pattern Search for pattern
  • ?pattern Search for pattern backwards
  • n Next match
  • N Previous match
  • ggn First match
  • GN Last match

i3

Focus window

  • Meta + h Focus left
  • Meta + j Focus down
  • Meta + k Focus up
  • Meta + l Focus right

Move window

  • Meta + Shift + h Move left
  • Meta + Shift + j Move down
  • Meta + Shift + k Move up
  • Meta + Shift + l Move right

Layout

  • Meta + e Tiled layout (default)
  • Meta + s Stacked layout
  • Meta + w Tabbed layout

Resizing

  • Meta + Right Click + Drag Resize
  • Meta + Alt + k Move top edge up
  • Meta + Alt + j Move top edge down
  • Meta + Alt + l Move right edge right
  • Meta + Alt + h Move right edge left
  • Meta + Alt + Shift + k Move bottom edge up
  • Meta + Alt + Shift + j Move bottom edge down
  • Meta + Alt + Shift + h Move left edge right
  • Meta + Alt + Shift + l Move left edge left

Workspaces

  • Meta + 0-9 Switch to workspace
  • Meta + Shift + 0-9 Move window to workspace

Floating

  • Meta + Shift + Space Toggle window floating
  • Meta + Space Toggle between floating/tiled layers
  • Meta + Drag Move floating window

Fullscreen

  • Meta + f Toggle fullscreen

Splitting

  • Meta + v Split vertically
  • Meta + h Split horizontally

Launching

  • Meta + Enter Open terminal
  • Meta + d Open dmenu launcher
  • Meta + c Open Chrome
  • Meta + Shift + q Kill window

i3 process

  • Meta + Shift + c Reload config file
  • Meta + Shift + r Restart i3
  • Meta + Shift + e Exit i3

urxvt

  • Ctrl + Alt + c Copy to clipboard
  • Ctrl + Alt + v Paste from clipboard
  • Alt + Shift + Drag Rectangular selection
  • Alt + s Search scrollback
